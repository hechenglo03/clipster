Clipster 自建分组功能产品方案

让用户按需创建自定义分组，跨分类组织常用剪贴内容

1. 背景与目标

当前 Clipster 的侧栏只有「分类」（文本 / 链接 / 图片 / 代码）和「收藏」两个维度。分类是按内容类型自动归类的，收藏是按「是否标星」归类的，二者都无法满足按项目、场景或主题组织剪贴内容的需求。

目标：引入「自建分组」能力，允许用户创建任意分组（如「工作日报素材」「面试题」「租房信息」），把条目自由归入一个或多个分组，并在侧栏快速切换查看。

2. 功能概述

  • 用户可以创建、重命名、改色、删除自定义分组。
  • 用户自由选择任意条目加入任意分组；一个条目可以同时属于多个分组。
  • 分组不会自动归类，是用户按项目 / 场景手动组织的「横向项目维度」，与自动分类相互独立。
  • 侧栏新增「我的分组」栏目，展示所有自建分组及其条目数。
  • 支持右键 / 快捷键 ⌘G 将选中条目加入分组。
  • 分组数据跟随 iCloud / 本地数据库同步（先本地，后续可扩展）。

3. 用户场景

场景 A：项目素材收集

设计师在做一个「品牌升级」项目，会把参考文案、配色链接、竞品截图、代码片段陆续复制到剪贴板。通过自建「品牌升级」分组，可以把跨分类的条目统一收纳，避免在全部列表里翻找。

场景 B：临时工作区

用户在处理一次线上故障，需要把错误日志、相关命令、沟通截图归到一起。创建一个「故障 2026-07-04」分组，处理完后再整组删除。

场景 C：知识库沉淀

开发者把常用的 Shell 命令、正则表达式、SQL 片段分别归入「常用命令」「正则」「SQL」分组，形成个人代码片段库。

4.2 创建分组

  1. 点击「我的分组」右侧的 + 新建。
  2. 弹出小浮层：输入分组名称、选择颜色（8 种预设）、选择图标（可选 Emoji 或纯色圆点）。
  3. 点击「创建」，分组立即出现在侧栏，条目数为 0。

4.3 把条目加入分组

  • 右键菜单：在条目上右键 → 「加入分组」→ 弹出分组列表，用户可自由勾选 1 个或多个分组。
  • 快捷键：选中条目后按 ⌘G，弹出分组选择浮层，自由选择分组。
  • 拖拽：长按条目拖到侧栏某个分组上松开，即可归入该分组（可选交互，v1 可不做）。

4.4 从分组移除

  • 在分组视图下，右键条目 → 「从本分组移除」，不会删除剪贴板条目本身。
  • 删除分组时，提示「分组内的条目将保留在全部列表中」。

4.5 分组管理

  • 右键侧栏分组 → 「重命名 / 改颜色 / 删除」。
  • 分组删除仅删除分组关系，不删除底层条目。
  • 支持拖拽排序（v2）。

5. 数据结构设计

5.1 新增表

表名字段说明
groups id INTEGER PRIMARY KEY
name TEXT NOT NULL
color TEXT （十六进制色值）
icon TEXT （Emoji 或 SF Symbol 名）
sort_order INTEGER
created_at INTEGER
updated_at INTEGER 		自建分组元数据
item_groups item_id INTEGER
group_id INTEGER
PRIMARY KEY (item_id, group_id) 		条目与分组的多对多关系

5.2 与现有表的关联

现有 items 表无需改动。item_groups 通过 item_id 外键关联。查询某分组下条目时：

SELECT i.* FROM items i
JOIN item_groups ig ON i.id = ig.item_id
WHERE ig.group_id = ?
ORDER BY i.created_at DESC;

6. 与现有功能的关系

维度自动/手动互斥/共存示例
分类自动（按内容类型）与分组共存一条文本可以同时属于「文本」分类和「品牌升级」分组
收藏手动（星标）与分组共存常用命令可以既标星又归入「常用命令」分组
分组手动（项目/场景）一个条目可在多个分组同一段文案可在「品牌升级」和「灵感」两个分组

7. 边界与异常处理

  • 同名分组：不允许创建同名分组，提示「分组名称已存在」。
  • 空分组：允许保留空分组，侧栏条目数显示 0。
  • 删除条目：删除剪贴板条目时，自动清理 item_groups 中的关联记录。
  • 删除分组：仅删除 groups 和 item_groups 关联，不删条目。
  • 搜索：在分组视图下搜索，只搜该分组内的条目；切回「全部」则恢复全局搜索。

8. 实现优先级建议

  • P0 分组增删改查 + 侧栏展示
  • P0 右键「加入分组」+ 多选
  • P1 快捷键 ⌘G 快速加入
  • P1 分组视图下搜索过滤
  • P2 拖拽入分组
  • P2 分组排序 / iCloud 同步

9. 收益总结

  • 从「时间线工具」升级为「剪贴板知识库」。
  • 解决长期使用后列表冗长、难以定位的问题。
  • 与现有的分类、收藏形成互补，不破坏现有逻辑。
  • 为后续「智能推荐」「工作区切换」等功能打下基础。
