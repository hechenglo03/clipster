Clipster 主面板 UI 设计稿

深色毛玻璃风格、侧栏 + 内容区两栏布局、列表与九宫格自适应切换

1. 设计目标

  • 一眼定位：侧栏提供分类、自建分组、收藏三个入口，内容区按当前维度过滤。
  • 高效操作：回车粘贴、数字键直达、方向键选择、快捷键分类切换。
  • 视觉降噪：深色背景 + 低饱和强调色，减少长期使用疲劳。
  • 内容自适应：文本/链接/代码用列表，图片用九宫格。

4. 侧栏设计

4.1 分区结构

从上到下依次为：

  1. 搜索框：始终置顶，激活后高亮边框。
  2. 全部：展示所有未删除条目，默认选中。
  3. 分类：文本、链接、图片、代码。
  4. 我的分组：用户自建分组，右侧有「+ 新建」入口。
  5. 收藏：常用、私密（后续可扩展）。

4.2 分组样式

  • 每个分组左侧使用颜色圆点（8px）或 Emoji 图标。
  • 右侧显示该分组下的条目数量。
  • hover 时背景变为 rgba(255,255,255,0.08)。
  • 选中时背景变为 rgba(91,140,255,0.14)，文字变白，数量变强调色。

5. 内容区设计

5.1 顶部标题栏

左侧显示「Clipster — N 条记录」，右侧显示当前选中的维度（如「文本分类」「品牌升级」）。高度 42px，底部 1px 分隔线。

5.2 列表视图（文本 / 链接 / 代码）

  • 每条高度 56px，内边距 10px 12px，圆角 10px。
  • 左侧 36px 图标区：分类 Emoji + 分类色背景。
  • 中间主内容：一行标题（截断），一行副标题（分类 + 来源/摘要）。
  • 右侧元信息：数字序号、相对时间。
  • Hover：背景变浅，边框出现。
  • 选中：蓝色背景 + 蓝色边框，序号变蓝。

5.3 图片九宫格视图

6. 搜索与过滤

  • 搜索框激活后，内容区实时过滤，高亮匹配关键词。
  • 搜索范围：当前维度（全部分类 / 当前分类 / 当前分组）。
  • 无结果时展示空状态：大图标 + 「未找到相关内容」。
  • 按 Esc 清空搜索并退出搜索框焦点。

8. 交互细节

操作行为
鼠标单击条目写入剪贴板并自动粘贴，面板关闭
按 ↩粘贴当前选中条目
按数字键 1-9直达并粘贴对应序号条目
按 ↑ / ↓上下移动选中
按 ⌘1-4切换分类（文本/链接/图片/代码）
按 ⌘G将选中条目加入分组
按 ⌘D收藏 / 取消收藏
按 ⌫删除当前选中条目
按 Esc关闭面板 / 清空搜索

9. 底部快捷键提示

底部固定 28px 高，左侧展示常用快捷键，右侧展示监听状态。提示颜色使用弱化色，不抢视觉焦点。

10. 后续可扩展

  • 多选批量操作（⌘Click 或 ⇧Click）。
  • 拖拽排序 / 拖拽入分组。
  • 列表与九宫格的视图切换按钮。
  • 主题切换（深色 / 浅色 / 跟随系统）。
  • 预览浮层（链接卡片、图片放大、代码高亮）。
